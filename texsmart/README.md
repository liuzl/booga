# TexSmart解析服务

## 准备

### 下载SDK

```sh
wget https://ai.tencent.com/ailab/nlp/texsmart/download/texsmart-sdk-0.1.1-s.zip
unzip texsmart-sdk-0.1.1-s.zip
mv texsmart-sdk-0.1.1-s/lib ./
mv texsmart-sdk-0.1.1-s/data ./
```

### 安装依赖

*需要提前安装好Anaconda3*

```sh
pip install fastapi
pip install uvicorn
```
